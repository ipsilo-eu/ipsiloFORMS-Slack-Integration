# ipsiloFORMS Slack Integration
A simple guide on how to use our Slack integration

# How to add an integration?



## Step 1: Login into your Slack workspace


## Step 2: Create a new channel
This is where you will recieve your notifications.


## Step 3: Access [api.slack.com/apps](https://api.slack.com/apps)


## Step 4: Create a new app
This can be done by clicking the **Create New App** button and then selecting the **From Scratch** option. Give your app a name and select the workspace it will be in.

## Step 5: Click on **Incoming Webhooks** from the **Features** section 

## Step 6: Add a new webhook
This can be done by clicking on the **Add New Webhook** button and selecting the desired channel for notifications when prompted.
